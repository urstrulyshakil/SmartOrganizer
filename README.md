# SmartOrganizer

**Boost your productivity and organize tasks efficiently.**

## Overview
SmartOrganizer helps you manage tasks, deadlines, and reminders all in one place.

## Features
- Task creation & management
- Priority labeling
- Calendar integration
- Notifications & reminders

## Tech Stack
- Python
- Tkinter / Streamlit
- SQLite
